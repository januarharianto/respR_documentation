respR – R Package Documentation

4 Quick start

respR has been designed to be simple to use, even for novice R users. Here we show you some examples of how data is analysed using various functions of the package.

4.1 Example data

Example data are available when respR is loaded: urchins.rd(), intermittent.rd(), zeb_intermittent.rd(), sardine.rd(), squid.rd(), flowthrough.rd(). Because lazy loading is implemented, the data can be called immediately. To view the list of available datasets and their descriptions, run the code:

data(package = "respR")

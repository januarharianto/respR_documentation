4 Import data

The import_file() function uses pattern recognititon to automatically recognise, clean and format file outputs from many commercially-availabe oxygen sensors. Currently supported systems include:

  • Pyroscience Firesting (pro, O2, GO2 and O2-Mini), and Pyro (software),
  • PRESENS OXY10 mini, PRESENS generic (micro, Microx, Fibox, OXY),
  • Aqualab MiniDOT,
  • Loligo Witrox 4, Witrox 1, and
  • Vernier LabQuest.

The function provides a minimalist way of handling raw data files – the user simply provides the file path, e.g.:

import_file("raw_output.txt")
